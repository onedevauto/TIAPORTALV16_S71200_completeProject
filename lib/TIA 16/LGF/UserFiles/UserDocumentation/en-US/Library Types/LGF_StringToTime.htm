LGF_StringToTime (FC / V3.0.1)

Author: Siemens Digital Industry

Short description

The function converts a variable of the data type String into a variable of the system data type Time.

Block Interface

LGF_StringToTime (FC)
String  timeValue Ret_Val  Time
    
   

Input parameter

IdentifierData typeDescription
timeValueStringTime to be converted as string
Example: `1D3H45M6S0MS`

Output parameter

IdentifierData typeDescription
Ret_ValTimeConverted time value
Example: `T#1D_3H_45M_6S`
