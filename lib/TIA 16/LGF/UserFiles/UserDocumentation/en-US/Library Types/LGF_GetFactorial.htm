LGF_GetFactorial (FC / V3.0.1)

Author: Siemens Digital Industry

Short description

The function calculates the faculty of a natural number (\mathbb{N}!) and returns the result.

The permissible value range of the input parameter naturalNumber is between 0 and 12, as 12 is the maximum factorial result fitting into a DInt type
